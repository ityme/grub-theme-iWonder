This project is a GRUB theme.
For Linux GRUB: 
  Please use `sudo ./install.sh` for automatic installation. This script will automatically create the theme files under `/boot/grub/themes/iWonder`.
  If you need to uninstall it, please use `sudo ./install.sh` for automatic uninstallation.

For Ventoy:
  Please first install it as a Linux GRUB theme on your Linux system, then copy the `/boot/grub/themes/iWonder` folder to your required Ventoy theme directory.
